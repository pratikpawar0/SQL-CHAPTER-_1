# 💻 SQL Company Database Project

## 📌 Overview

This project demonstrates basic to intermediate SQL concepts by creating a relational database with 3 tables:

* Employees
* Departments
* Projects

## 🧱 Database Structure

* **employees** → Stores employee details
* **departments** → Stores department names
* **projects** → Stores project assignments

## ⚙️ Concepts Used

* SELECT, WHERE, ORDER BY, LIMIT
* GROUP BY & HAVING
* Aggregate Functions (MAX, MIN, AVG)
* INNER JOIN & LEFT JOIN
* 3 Table JOIN 🔥
* Subqueries

## 📊 Sample Queries

* Find employees with above-average salary
* Get department-wise employee count
* Count projects per employee

## 🚀 How to Run

1. Open MySQL / SQL Workbench
2. Run the SQL script
3. Execute queries step by step
